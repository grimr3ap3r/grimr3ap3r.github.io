Smart Parking solution enables the parking lot management to automate the process of assigning a parking space to their customers.

Adaptive Traffic Control
Jan. 2018
Python
TensorFlow,  OpenCV,  NumPy
CMS Computers
Traffic signals often do not take into account the number of cars at a particular junction. Adaptive Traffic Control attempts to fix this problem using computer vision. With cameras already installed on the junction, number of cars are detected as well as tracked to avoid multiple counts of the same car. This count is then sent to traffic signal controller which manipulates the signal timing accordingly. For this project, SSD Mobilenet model was used and deployed on the embedded board - Nvidia Jetson TX2.
